Manual Setup Instructions

Since we need admin access to create storage buckets, please follow these steps in your Supabase Dashboard:

Step 1: Go to your Supabase project dashboard at:
https://supabase.com/dashboard/project/wxvcmvacajexdtcbgdva
Step 2: Click on "Storage" in the left sidebar
Step 3: Click "New bucket" button
Step 4: Fill in the following details:
  • Bucket name: avatars
  • Public bucket: âœ… Yes (check this)
  • File size limit: 5 MB
  • Allowed MIME types: image/jpeg, image/png, image/gif, image/webp
Step 5: Click "Create bucket"
Step 6: After creating the bucket, go to the SQL Editor and run this to set up policies:
-- Allow public access to view avatars CREATE POLICY "Avatar images are publicly accessible" ON storage.objects FOR SELECT USING (bucket_id = 'avatars'); -- Allow authenticated users to upload their own avatar CREATE POLICY "Users can upload their own avatar" ON storage.objects FOR INSERT WITH CHECK (bucket_id = 'avatars' AND auth.uid() IS NOT NULL); -- Allow users to update their own avatar CREATE POLICY "Users can update their own avatar" ON storage.objects FOR UPDATE USING (bucket_id = 'avatars' AND auth.uid() IS NOT NULL); -- Allow users to delete their own avatar CREATE POLICY "Users can delete their own avatar" ON storage.objects FOR DELETE USING (bucket_id = 'avatars' AND auth.uid() IS NOT NULL);
